## 介绍

`MindCanvas` 是一款制作思维导图的软件。使用 `MindCanvas`，你可以通过添加点、线和墨迹轻松创造出自己的思维导图，并导出或分享。
<p align="center">
<img width="80%" src="https://s3.bmp.ovh/imgs/2021/10/1213ae9fcf532bfd.png"/>
</p>

## 功能

> 使用点、线和墨迹制作思维导图  
> 导出、分享、加密思维导图  
> 灵活的样式和浅色、深色两种主题  
> 无限大的画布  
> 支持多国语言
